A modern web application built with React and JavaScript, featuring dynamic UI components and state management with Redux. The project uses HTML and CSS for responsive layout and styling.

It integrates API requests through an Express backend, enabling seamless data handling and communication between client and server.

Various React hooks such as useState, useEffect, and custom form handling hooks are utilized to manage component state, side effects, and user input efficiently.

This project demonstrates practical implementation of front-end architecture, state management, and client-server interaction in a real-world scenario.
